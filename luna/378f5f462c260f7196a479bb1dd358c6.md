いかがだったでしょうか？<br/>

<br/>

<br/>
定期公演があったので、<br/>

<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>

<br/>
「中々開かない踏切みたいに<br/>
<br/>
待たされ続けて」<br/>

私推しじゃなくても、<br/>
<br/>
「うん、うん。」って顔で<br/>

<br/>
<br/>
<br/>

<br/>
<br/>
<br/>

ちなみに四条推しの方とここで目が合うと、<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
やっぱり踊っている際に、<br/>
<br/>
先輩メンバーの背中が見えるのが、<br/>

両脇に先輩メンバーがいるのが、<br/>
<br/>
すごく安心します。<br/>

<br/>
後ろにいる私からＶ字に広がるように、<br/>
<br/>

<br/>
<br/>
<br/>
<br/>
<br/>
<br/>
特に、にゃん先輩の背中が見えて、<br/>
<br/>

<br/>
なんて思ったりもしました。笑笑<br/>

<br/>

<br/>
「願いの眼差し」でした。<br/>

ファンの皆様や、先輩方のおかげで、<br/>

我ながら一曲目から、しっかり決められた気がします。<br/>
